2. 이클립스 (IDE) 설치
	STS는 스프링전용 이클립스다.
	나중에 스프링수업때 STS를 또 깔고, 또 환경세팅하고 해야되기 때문에 자바할때부터 바로 STS설치.
	구글에서 STS 쳐서 https://spring.io/tools 에서 운영체제에 맞는거 다운로드
	더블클릭해서 압축풀고 원하는 경로로 이동.
	SpringToolSuite4.ini 파일 열어서 메모리 설정  최소는 1/8. 최대는 1/4.
					-vm 밑의줄에 참조할 java홈경로의 bin까지 주소 붙여넣기
	exe파일 더블클릭으로 실행. workspace설정 자기가 하고싶은곳으로 해주고 체크해주고 launch
	왼쪽에서 create java project 버튼 클릭. 모듈체크 해제하고 finish
	src폴더 우클릭 - new - class - 이름정하고 - main함수 체크
	클래스파일의 main함수 안에서 System.out.println("hello world");
	우클릭 - run as - Java Application  하면 콘솔에 hello world찍힌다.
	
	main함수는 한프로젝트에 하나밖에 없는 함수다.
	프로젝트가 실행되면 main함수가 실행된다.
	syso 누르고 ctrl space 누르면 자동완성됨.
	ctrl f11 : 제일 최근에 실행했던 run 다시 실행
	ctrl shift r : 파일이름으로 파일 자동 찾기
	
	변수 선언, 데이터타입
	함수 만들기 연습

	프로젝트 내 workspace로 가져오는 법 : file - import - genral - Existing project into workspace(이클립스로 export한 파일 쉽게 가져올때)
